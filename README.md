# 인건비 신고 자동화 (원천세 · 간이지급명세서)

세무대리인이 **원천세(홈택스)** + **지방소득세 특별징수(위택스)** + **간이지급명세서(홈택스)**를
신고·서류처리하는 과정을 자동화하는 Windows 데스크톱 프로그램.

- 스택: **Python + Playwright(Chromium) + Tkinter**
- 브라우저: 영구 프로필 Chromium — 로그인 한 번이면 세션 유지 (`.profile`)
- 변환파일은 **세무사랑**에서 미리 제작. 이 프로그램은 그 이후(신고서 제출 → 납부서·서류 출력)만 담당.
- 설계 배경: [docs/decisions.md](docs/decisions.md) · 패턴 모음: [reference/AUTOMATION_PATTERNS.md](reference/AUTOMATION_PATTERNS.md)

> 같은 사무소의 성공한 **양도세 자동화(`yangdo_auto`)와 동일 스택·패턴·엔진**.
> 파일변환신고/clipreport/OZ뷰어 기계장치는 yangdo_auto에서 라이브 검증된 코드를 그대로 가져옴.

## 흐름 — Phase 단위 독립 (선택/연속 실행)

| 순서 | Phase | 메모 |
|---|---|---|
| 1 | 위택스 특별징수 파일신고 | 납부서 가상계좌 지연 대비 먼저 |
| 2 | 홈택스 원천세 파일변환신고 | 세무사랑 변환파일 |
| 3 | 홈택스 간이지급명세서 제출 | 여러 파일이면 `;`로 구분 |
| 4 | 홈택스 접수증·신고서 출력 | 옵션 |
| 5 | 위택스 신고서 출력 | 옵션 |
| 6 | 홈택스 납부서 출력 | 사업자번호로 신고내역 조회 |
| 7 | 위택스 납부서 출력 | 가상계좌 생성 대기 후 |

각 사이트 로그인은 사용자가 직접, 이후는 자동. 각 phase는 진입 시 홈으로 하드 리셋(goto)
→ 앞 phase의 상태와 무관하게 독립 실행 가능.

## 개발 시작

```powershell
pip install -r requirements.txt
python -m playwright install chromium
python gui.py
```
