인건비(원천세·간이지급명세서) 신고 자동화 — 사용 안내
====================================================

[준비물]
- Windows 10/11 (64비트)
- 홈택스 / 위택스 로그인 수단(공동인증서 등) — 로그인은 직접 하셔야 합니다(자동화 안 됨)
- 세무사랑으로 만든 변환파일
    · 홈택스 원천세      : *.01 / *.enc
    · 간이지급명세서     : 종류별 변환파일 (근로소득·사업소득 등)
    · 위택스 특별징수    : 회계파일 (업체당 1개)
- 세무사랑 전자신고 파일 비밀번호
- 'Microsoft Print to PDF' 프린터 (Windows 기본 제공) — PDF 저장 시 사용

[실행]
1. 압축을 풀고 IngunbiAuto.exe 를 실행합니다. (설치 불필요)
   ※ 첫 실행 시 'Windows 보안 경고'가 뜨면 '추가 정보' → '실행'.
2. 실행할 단계를 켜고, 입력 정보(파일·사업자등록번호·비밀번호 등)를 채웁니다.
3. [시작]을 누르면 홈택스·위택스 창이 열립니다. 각 사이트에 직접 로그인하세요.
   → 로그인이 확인되면 자동으로 진행됩니다.
4. 실행이 끝나도 브라우저는 유지됩니다 — 다음 업체 파일로 바꿔서 바로 [시작]을
   누르면 재로그인 없이 이어서 처리됩니다.

[단계]
 ① 위택스 특별징수 파일신고   ② 홈택스 원천세 파일변환신고   ③ 홈택스 간이지급명세서 제출
 ④ 홈택스 접수증·신고서 출력   ⑤ 위택스 신고서 출력
 ⑥ 홈택스 납부서 출력          ⑦ 위택스 납부서 출력
 - 기본 선택은 ①②③⑥⑦ 입니다. 접수증·신고서가 필요하면 ④⑤를 켜세요.
 - 납부세액이 없는 업체는 ⑥⑦을 켜도 '납부서 없음'으로 정상 종료됩니다.
 - 위택스 가상계좌는 신고 후 몇 분 뒤 생성되므로, ⑦ 시작 전 '위택스 납부서 대기'(분)만큼 기다립니다.

[입력]
 - 간이지급명세서: 종류(드롭다운)를 고르고 해당 변환파일을 첨부 — 종류마다 개별 제출됩니다.
   여러 종류면 '＋ 간이지급명세서 추가'로 행을 늘리세요.
 - 사업자등록번호: 납부서·접수증 출력 시 신고내역 조회에 사용 (하이픈 없이 10자리)
 - 납부기한(선택): 홈택스 납부서 PDF 파일명에 쓸 날짜 — 입력한 형식 그대로 파일명에 들어가고,
   다음 실행에도 유지됩니다(월초에 한 번만 입력). 비우면 자동 계산/추출.
 - 파일 비밀번호: 한 번 입력하면 다음 실행에 자동으로 채워집니다.

[주의]
 - ①②③은 실제 '제출'까지 진행됩니다(비가역). 검증만 원하면 해당 단계를 끄고 쓰세요.
 - 로그인 정보는 저장하지 않습니다. (같은 실행 중에는 세션이 유지됩니다)
 - 데이터(로그인 세션)는 %LOCALAPPDATA%\IngunbiAuto 에 저장됩니다.
 - [중단]은 즉시 멈춥니다. 신고 단계 도중 중단했다면 홈택스/위택스에서 접수 여부를 확인하세요.
